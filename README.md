# BEP Field Lab v2

**Research-grade interactive tool for testing Belonging–Energy–Prediction (BEP) field dynamics.**

Upload any CSV or Excel file, map multiple proxies to Energy / Belonging / Prediction, run composite scoring, compare against single-variable baselines, inspect field states, and get an instant exploratory interpretation report.

### Live public version
→ **https://bepframework.github.io/bep-field-lab/** (once GitHub Pages is enabled)

### Features
- Multi-proxy composites (average any number of columns)
- Baseline vs BEP model comparison + signed improvement delta
- Field quadrant map, correlation heatmap, group & time trends
- Field state classification (Higher / Mixed / Lower)
- Exportable researcher-friendly report (includes your notes)
- Fully offline — single HTML file, no server or login required

### How to use
1. Open `index.html` in any browser (or visit the live link above)
2. Drop your dataset
3. Select proxies → Run analysis
4. Export report for your paper or lab notebook
